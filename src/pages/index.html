How It Works

From question to answer in milliseconds.

Ask a Question

Type your question or select a quick prompt in the chat widget. You can also call the REST API directly from your code or AI IDE.

Cloudflare Workers AI

Your request is routed to the nearest Cloudflare edge node, where a Llama 3.1 model processes your message with BLT-specific context.

Agent Orchestration

For complex tasks, Byte dispatches sub-tasks (FAQ lookup, scan analysis, onboarding flow) and assembles a comprehensive response.

Instant Response

Results are returned as structured JSON or rendered in the chat UI – ready to act on immediately.

API Reference

Integrate Byte into your own tools via the REST API or the built-in MCP server.

Method Endpoint Description Body / Params
GET /api/health Health check – verify the service is running —
POST /api/chat Send a message to the FAQ / onboarding agent { "message": "...", "history": [] }
POST /api/scan Request an AI-generated security scan for a URL { "url": "https://...", "scan_type": "quick|full" }
GET /api/mcp Retrieve the MCP tool manifest (for AI IDE integration) —
POST /api/mcp Invoke an MCP tool (chat, scan_url, get_onboarding_guide) { "tool": "chat", "params": { "message": "..." } }
